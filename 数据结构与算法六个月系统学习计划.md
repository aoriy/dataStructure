# 数据结构与算法 — 六个月系统学习计划

## 总览

| 阶段 | 时间 | 主题 | 目标 |
|------|------|------|------|
| 基础 | 第1-2月 | 基础数据结构与复杂度分析 | 掌握数组、链表、栈、队列、哈希表、字符串 |
| 进阶 | 第3-4月 | 树、图与核心算法思想 | 掌握二叉树、堆、图、排序、二分、贪心、动态规划 |
| 冲刺 | 第5-6月 | 高级专题与综合刷题 | 回溯、滑动窗口、并查集、高频题、模拟面试 |

## 第1个月 — 基础数据结构（上）

  - LeetCode 26（删除有序数组重复项）

- **自学**：复习已完成的文件 `二分查找完全学习指南.md`

### 第2周：链表
- **单链表、双链表、循环链表**
- **经典操作**：反转、合并、找环
- **练习题**：
  - LeetCode 206（反转链表）
  - LeetCode 141（环形链表）
  - LeetCode 21（合并两个有序链表）
  - LeetCode 160（相交链表）

### 第3周：栈与队列
- **栈**：先进后出，单调栈
- **队列**：先进先出，双端队列，优先队列
- **练习题**：
  - LeetCode 20（有效括号）
  - LeetCode 155（最小栈）

- **自学**：复习 `单调栈完全学习指南.md`

### 第4周：综合复习 + 小测验
- 重做前3周的错题
- 限时完成 3-5 道 LeetCode 简单题
- 整理笔记到 Excel 错题集

### 推荐刷题顺序
1. LeetCode 热题 100（Hot 100）
2. LeetCode 精选 TOP 面试题
3. 剑指 Offer（专项突破）

### 每日建议
- **工作日**：1-2 小时（1 道新题 + 复习 1 道旧题）
- **周末**：3-4 小时（专题学习 + 总结笔记）
- 每道题先独立思考 15-20 分钟，再看题解
- 用错题集 Excel 记录每道题的思路、坑点、模板
