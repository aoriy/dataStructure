# AGENTS.md — 数据结构与算法学习笔记

## 项目本质

个人面试准备（华为 OD）的学习笔记仓库，**不含可编译/运行的代码**。所有 Python/C++ 代码片段均嵌入 Markdown 内作为示例。

## 文件命名约定

- `完全学习指南.md` — 专题系统教程（二叉树、哈希、滑动窗口等）
- `_题解分析.md` / `_代码分析.md` — 单道算法题的详细解析
- `.xlsx` — 错题集与考试计划表

## 已知约束

- 无构建脚本、无测试框架、无包管理器、无 CI
- `.gitignore` 排除了 `.uploads/` 和 `(已自动还原).xlsx`
- HTML 文件存在 LF/CRLF 混用，Git 会发出警告，不影响功能
- 所有 Markdown 文件中代码块以 Python 为主，少量 C++
- 文件名含中文，在终端中使用引号包裹或 PowerShell Tab 补全
