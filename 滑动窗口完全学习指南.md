# 第一部分：滑动窗口核心概念

## 1.1 什么是滑动窗口

### 定义

**滑动窗口**是一种在数组/字符串上维护一个**可移动的区间**的算法技巧。通过调整区间的左右边界，高效地解决**子数组/子串**相关问题。

### 直觉理解

想象一个**滑动的框**在数组上移动：

```
数组：[1, 3, -1, -3, 5, 3, 6, 7]
窗口大小：3

初始状态：
[1  3  -1] -3  5  3  6  7   ← 窗口覆盖 [1, 3, -1]

向右滑动一步：
 1 [3  -1  -3] 5  3  6  7   ← 窗口覆盖 [3, -1, -3]

再滑动一步：
 1  3 [-1  -3  5] 3  6  7   ← 窗口覆盖 [-1, -3, 5]

...继续滑动...
```

### 核心思想

> **"维护一个窗口，通过移动左右边界来遍历所有可能的子数组/子串，同时利用已计算的信息避免重复计算"**

### 为什么滑动窗口高效？

**暴力解法**：枚举所有子数组，每个子数组重新计算 → O(n²) 或 O(n³)

**滑动窗口**：
- 窗口移动时，只处理**新增元素**和**移出元素**
- 利用之前的结果，避免重复计算
- 时间复杂度通常降为 **O(n)**

## 1.2 滑动窗口的分类

### 按窗口大小分类

| 类型 | 特点 | 典型问题 |
|------|------|---------|
| **定长滑动窗口** | 窗口大小固定为 k | 滑动窗口最大值、长度为 k 的子数组和 |
| **不定长滑动窗口** | 窗口大小动态变化 | 最长无重复子串、最小覆盖子串 |

### 按窗口变化方式分类

| 类型 | 窗口变化 | 典型问题 |
|------|---------|---------|
| **窗口扩大型** | 只增不减，求最大长度 | 最长无重复子串 |
| **窗口缩小型** | 只减不增，求最小长度 | 最小覆盖子串 |
| **窗口伸缩型** | 可增可减 | 和为 K 的子数组个数 |

### 图解三种类型

```
窗口扩大型（求最长）：
初始：[a] b c a b c
扩大：[a b] c a b c
扩大：[a b c] a b c  ← 遇到重复，收缩
收缩：a [b c] a b c
扩大：a [b c a] b c
...

窗口缩小型（求最小）：
初始：[a d o b e c] o d e b a n c  ← 包含所有目标字符
缩小：a [d o b e c] o d e b a n c  ← 尝试缩小
缩小：a d [o b e c] o d e b a n c
...

窗口伸缩型（求满足条件的个数）：
初始：[1] 1 1
扩大：[1 1] 1
扩大：[1 1 1]  ← 和 = 3，满足
收缩：1 [1 1]  ← 和 = 2，不满足
扩大：1 [1 1 1]  ← 和 = 3，满足
...
```

## 1.3 滑动窗口的适用条件

### 什么时候可以用滑动窗口？

| 条件 | 说明 |
|------|------|
| ✅ **连续子数组/子串** | 问题涉及连续区间 |
| ✅ **单调性** | 窗口扩大时，某个指标单调变化（如和增加、长度增加） |
| ✅ **可增量更新** | 窗口移动时，可以快速更新结果（如加减元素） |
| ❌ **不连续子序列** | 需要用 DP 或回溯 |
| ❌ **无单调性** | 无法确定窗口何时扩大/缩小 |

### 判断流程

```
问题涉及"子数组"或"子串"？
├── 否 → 不适合滑动窗口
└── 是
    ├── 窗口大小是否固定？
    │   ├── 是 → 定长滑动窗口
    │   └── 否
    │       ├── 求"最长/最大"？
    │       │   └── 是 → 窗口扩大型
    │       ├── 求"最短/最小"？
    │       │   └── 是 → 窗口缩小型
    │       └── 求"满足条件的个数"？
    │           └── 是 → 窗口伸缩型
```

## 1.4 滑动窗口的本质

### 双指针的特例

滑动窗口本质上是**双指针**的一种特殊形式：
- **left 指针**：窗口左边界
- **right 指针**：窗口右边界
- 窗口大小 = right - left + 1

### 与其他双指针的区别

| 类型 | 指针移动方式 | 典型问题 |
|------|------------|---------|
| **对撞指针** | 两端向中间移动 | 两数之和、反转数组 |
| **快慢指针** | 速度不同 | 环检测、找中点 |
| **滑动窗口** | 同向移动，维护区间 | 子数组/子串问题 |

### 时间复杂度分析

| 操作 | 复杂度 | 说明 |
|------|--------|------|
| right 右移 | O(n) | 最多移动 n 次 |
| left 右移 | O(n) | 最多移动 n 次 |
| **总计** | **O(n)** | 每个元素最多被访问 2 次 |

# 第二部分：滑动窗口代码模板

## 2.1 定长滑动窗口模板

### 适用场景

- 窗口大小固定为 k
- 求**每个窗口**的最大值/最小值/平均值等

### 标准模板

```python
def fixed_sliding_window(nums, k):
    n = len(nums)
    if n < k:
        return []

    # 初始化：计算第一个窗口的结果
    window_sum = sum(nums[:k])
    result = [window_sum]

    # 滑动窗口
    for i in range(k, n):
        # 移出左边元素，加入右边元素
        window_sum = window_sum - nums[i - k] + nums[i]
        result.append(window_sum)

    return result
```

### 图解过程

```
数组：[1, 3, -1, -3, 5, 3, 6, 7]，k = 3

初始窗口：[1, 3, -1]，和 = 3

第1次滑动：
  移出 1，加入 -3
  新窗口：[3, -1, -3]，和 = 3 - 1 + (-3) = -1

第2次滑动：
  移出 3，加入 5
  新窗口：[-1, -3, 5]，和 = -1 - 3 + 5 = 1

...继续滑动...
```

### 经典应用：滑动窗口最大值

```python
from collections import deque

def max_sliding_window(nums, k):
    """使用单调队列求滑动窗口最大值"""
    n = len(nums)
    if n < k:
        return []

    result = []
    dq = deque()  # 存储索引，维护单调递减队列

    for i in range(n):
        # 移除超出窗口的元素
        while dq and dq[0] < i - k + 1:
            dq.popleft()

        # 维护单调递减：移除所有比当前元素小的
        while dq and nums[dq[-1]] < nums[i]:
            dq.pop()

        dq.append(i)

        # 窗口形成后，记录最大值
        if i >= k - 1:
            result.append(nums[dq[0]])

    return result
```

## 2.2 不定长滑动窗口模板（可变窗口）

### 模板一：窗口扩大型（求最长）

```python
def longest_window(s):
    left = 0
    max_len = 0
    window = {}  # 记录窗口内元素

    for right in range(len(s)):
        # 扩大窗口：加入右边界元素
        window[s[right]] = window.get(s[right], 0) + 1

        # 当窗口不满足条件时，收缩左边界
        while 不满足条件:
            window[s[left]] -= 1
            if window[s[left]] == 0:
                del window[s[left]]
            left += 1

        # 更新结果
        max_len = max(max_len, right - left + 1)

    return max_len
```

### 模板二：窗口缩小型（求最短）

```python
def shortest_window(s, t):
    from collections import Counter

    need = Counter(t)  # 需要的字符
    missing = len(t)   # 缺失的字符数
    left = 0
    min_len = float('inf')
    start = 0

    for right in range(len(s)):
        # 扩大窗口
        if s[right] in need:
            if need[s[right]] > 0:
                missing -= 1
            need[s[right]] -= 1

        # 当窗口满足条件时，尝试缩小
        while missing == 0:
            if right - left + 1 < min_len:
                min_len = right - left + 1
                start = left

            # 移出左边界元素
            if s[left] in need:
                need[s[left]] += 1
                if need[s[left]] > 0:
                    missing += 1
            left += 1

    return s[start:start+min_len] if min_len != float('inf') else ""
```

### 模板三：窗口伸缩型（求满足条件的个数）

```python
def count_subarrays(nums, k):
    """求和 >= k 的子数组个数"""
    count = 0
    left = 0
    window_sum = 0

    for right in range(len(nums)):
        # 扩大窗口
        window_sum += nums[right]

        # 当满足条件时，统计并尝试收缩
        while window_sum >= k:
            count += len(nums) - right  # 以 left 开头的所有子数组都满足
            window_sum -= nums[left]
            left += 1

    return count
```

## 2.3 两种模板对比

| 对比维度 | 定长滑动窗口 | 不定长滑动窗口 |
|---------|-------------|---------------|
| **窗口大小** | 固定为 k | 动态变化 |
| **指针移动** | right 每次移动 1 步 | right 和 left 都可能移动 |
| **循环结构** | 单层 for 循环 | for + while 嵌套 |
| **时间复杂度** | O(n) | O(n)（均摊分析） |
| **典型问题** | 滑动窗口最大值 | 最长无重复子串 |

## 3.1 六大核心技巧

### 技巧一：使用哈希表记录窗口状态

```python
# 记录字符出现次数
window = {}
window[char] = window.get(char, 0) + 1

# 记录字符最后出现位置
last_seen = {}
last_seen[char] = right

# 记录字符集合
window_set = set()
window_set.add(char)
```

### 技巧二：使用变量维护窗口属性

```python
# 维护窗口和
window_sum += nums[right]  # 扩大
window_sum -= nums[left]   # 缩小

# 维护窗口最大值/最小值（单调队列）
# 维护满足条件的元素个数
```

### 技巧三：使用计数器简化判断

```python
from collections import Counter

need = Counter(t)  # 目标字符计数
have = Counter()   # 当前窗口计数

# 判断是否满足条件
def is_valid():
    for char, count in need.items():
        if have[char] < count:
            return False
    return True
```

### 技巧四：使用 missing 变量优化

```python
# 代替每次遍历 need 判断是否满足
missing = len(t)  # 缺失的字符数

# 当 missing == 0 时，窗口满足条件
# 当 missing > 0 时，窗口不满足条件
```

### 技巧五：单调队列处理最值

```python
from collections import deque

dq = deque()  # 单调递减队列，队首是最大值

# 移除超出窗口的元素
while dq and dq[0] < left:
    dq.popleft()

# 维护单调性
while dq and nums[dq[-1]] < nums[right]:
    dq.pop()
dq.append(right)

# 获取当前窗口最大值
max_val = nums[dq[0]]
```

### 技巧六：前缀和 + 滑动窗口

```python
# 前缀和数组
prefix = [0] * (len(nums) + 1)
for i in range(len(nums)):
    prefix[i + 1] = prefix[i] + nums[i]

# 窗口和 = prefix[right + 1] - prefix[left]
window_sum = prefix[right + 1] - prefix[left]
```

## 3.2 常见陷阱与易错点

### 陷阱一：窗口收缩条件错误

```python
# ❌ 错误：收缩条件写错
while window[s[right]] > 1:  # 应该是 window[s[left]]
    ...

# ✅ 正确：收缩条件基于左边界
while window[s[left]] > 1:
    window[s[left]] -= 1
    left += 1
```

### 陷阱二：更新结果的时机错误

```python
# ❌ 错误：在收缩前更新
max_len = max(max_len, right - left + 1)
while 不满足条件:
    left += 1

# ✅ 正确：在收缩后更新
while 不满足条件:
    left += 1
max_len = max(max_len, right - left + 1)
```

### 陷阱三：忘记处理窗口边界

```python
# ❌ 错误：窗口大小计算错误
window_size = right - left  # 少了 +1

# ✅ 正确：
window_size = right - left + 1
```

### 陷阱四：哈希表删除时机错误

```python
# ❌ 错误：直接删除
del window[s[left]]  # 可能键不存在

# ✅ 正确：先减后判断
window[s[left]] -= 1
if window[s[left]] == 0:
    del window[s[left]]
```

### 陷阱五：定长窗口的边界处理

```python
# ❌ 错误：从索引 0 开始记录结果
for right in range(n):
    # 处理...
    result.append(...)  # 窗口还没形成

# ✅ 正确：窗口形成后才开始记录
for right in range(n):
    # 处理...
    if right >= k - 1:  # 窗口已形成
        result.append(...)
```

### 陷阱六：空输入处理

```python
# ❌ 错误：没有处理空输入
def sliding_window(s):
    left = 0
    for right in range(len(s)):
        ...

# ✅ 正确：先检查空输入
def sliding_window(s):
    if not s:
        return 0
    ...
```

## 3.3 滑动窗口 vs 其他算法

### 滑动窗口 vs 双指针

| 对比维度 | 滑动窗口 | 双指针 |
|---------|---------|--------|
| **指针移动** | 同向移动 | 可同向、可相向 |
| **维护内容** | 一个区间 | 两个独立位置 |
| **典型问题** | 子数组/子串 | 两数之和、反转 |
| **关系** | 滑动窗口是双指针的特例 | — |

### 滑动窗口 vs 前缀和

| 对比维度 | 滑动窗口 | 前缀和 |
|---------|---------|--------|
| **核心操作** | 增量更新 | 预计算 + 区间查询 |
| **适用场景** | 动态窗口 | 静态区间求和 |
| **时间复杂度** | O(n) | O(n) 预处理 + O(1) 查询 |
| **组合使用** | 可以结合 | — |

### 滑动窗口 vs 动态规划

| 对比维度 | 滑动窗口 | 动态规划 |
|---------|---------|---------|
| **子问题关系** | 连续区间 | 可不连续 |
| **状态转移** | 窗口移动 | 状态方程 |
| **空间复杂度** | O(1) ~ O(n) | O(n) ~ O(n²) |
| **适用条件** | 单调性 | 最优子结构 |

## 4.1 入门级题目

### 🟢 LeetCode 209 - 长度最小的子数组 ⭐⭐

**难度**：中等 | **类型**：不定长窗口（缩小型）

**题目**：给定一个正整数数组 `nums` 和一个正整数 `target`，找出数组中满足其和 ≥ target 的最小长度子数组。

```python
def minSubArrayLen(target, nums):
    left = 0
    window_sum = 0
    min_len = float('inf')

    for right in range(len(nums)):
        window_sum += nums[right]

        while window_sum >= target:
            min_len = min(min_len, right - left + 1)
            window_sum -= nums[left]
            left += 1

    return min_len if min_len != float('inf') else 0
```

### 🟢 LeetCode 3 - 无重复字符的最长子串 ⭐⭐

**难度**：中等 | **类型**：不定长窗口（扩大型）

```python
def lengthOfLongestSubstring(s):
    left = 0
    max_len = 0
    window = set()

    for right in range(len(s)):
        while s[right] in window:
            window.remove(s[left])
            left += 1

        window.add(s[right])
        max_len = max(max_len, right - left + 1)

    return max_len
```

### 🟢 LeetCode 219 - 存在重复元素 II ⭐

**难度**：简单 | **类型**：定长窗口

```python
def containsNearbyDuplicate(nums, k):
    window = set()

    for right in range(len(nums)):
        if nums[right] in window:
            return True

        window.add(nums[right])

        if len(window) > k:
            window.remove(nums[right - k])

    return False
```

### 🟢 LeetCode 1456 - 定长子串中元音的最大数目 ⭐

**难度**：中等 | **类型**：定长窗口

```python
def maxVowels(s, k):
    vowels = set('aeiou')
    count = 0
    max_count = 0

    # 初始化第一个窗口
    for i in range(k):
        if s[i] in vowels:
            count += 1
    max_count = count

    # 滑动窗口
    for i in range(k, len(s)):
        if s[i - k] in vowels:
            count -= 1
        if s[i] in vowels:
            count += 1
        max_count = max(max_count, count)

    return max_count
```

### 🟡 LeetCode 76 - 最小覆盖子串 ⭐⭐⭐

**难度**：困难 | **类型**：不定长窗口（缩小型）

```python
def minWindow(s, t):
    from collections import Counter

    need = Counter(t)
    missing = len(t)
    left = 0
    min_len = float('inf')
    start = 0

    for right in range(len(s)):
        if s[right] in need:
            if need[s[right]] > 0:
                missing -= 1
            need[s[right]] -= 1

        while missing == 0:
            if right - left + 1 < min_len:
                min_len = right - left + 1
                start = left

            if s[left] in need:
                need[s[left]] += 1
                if need[s[left]] > 0:
                    missing += 1
            left += 1

    return s[start:start+min_len] if min_len != float('inf') else ""
```

### 🟡 LeetCode 438 - 找到字符串中所有字母异位词 ⭐⭐

**难度**：中等 | **类型**：定长窗口 + 哈希

```python
def findAnagrams(s, p):
    from collections import Counter

    result = []
    p_count = Counter(p)
    window_count = Counter(s[:len(p)])

    if window_count == p_count:
        result.append(0)

    for i in range(len(p), len(s)):
        # 移出左边元素
        left_char = s[i - len(p)]
        window_count[left_char] -= 1
        if window_count[left_char] == 0:
            del window_count[left_char]

        # 加入右边元素
        window_count[s[i]] += 1

        if window_count == p_count:
            result.append(i - len(p) + 1)

    return result
```

### 🟡 LeetCode 567 - 字符串的排列 ⭐⭐

**难度**：中等 | **类型**：定长窗口

```python
def checkInclusion(s1, s2):
    from collections import Counter

    if len(s1) > len(s2):
        return False

    s1_count = Counter(s1)
    window_count = Counter(s2[:len(s1)])

    if s1_count == window_count:
        return True

    for i in range(len(s1), len(s2)):
        left_char = s2[i - len(s1)]
        window_count[left_char] -= 1
        if window_count[left_char] == 0:
            del window_count[left_char]

        window_count[s2[i]] += 1

        if s1_count == window_count:
            return True

    return False
```

### 🟡 LeetCode 239 - 滑动窗口最大值 ⭐⭐⭐

**难度**：困难 | **类型**：定长窗口 + 单调队列

```python
from collections import deque

def maxSlidingWindow(nums, k):
    result = []
    dq = deque()  # 存储索引，单调递减

    for i in range(len(nums)):
        # 移除超出窗口的元素
        while dq and dq[0] < i - k + 1:
            dq.popleft()

        # 维护单调递减
        while dq and nums[dq[-1]] < nums[i]:
            dq.pop()

        dq.append(i)

        # 窗口形成后记录结果
        if i >= k - 1:
            result.append(nums[dq[0]])

    return result
```

### 🟡 LeetCode 904 - 水果成篮 ⭐⭐

**难度**：中等 | **类型**：不定长窗口（扩大型）

```python
def totalFruit(fruits):
    left = 0
    max_fruits = 0
    basket = {}  # 篮子：水果类型 -> 数量

    for right in range(len(fruits)):
        basket[fruits[right]] = basket.get(fruits[right], 0) + 1

        # 超过两种水果，收缩窗口
        while len(basket) > 2:
            basket[fruits[left]] -= 1
            if basket[fruits[left]] == 0:
                del basket[fruits[left]]
            left += 1

        max_fruits = max(max_fruits, right - left + 1)

    return max_fruits
```

**难度**：困难 | **类型**：不定长窗口 + 转化

```python
def subarraysWithKDistinct(nums, k):
    """恰好 K 个 = 最多 K 个 - 最多 K-1 个"""

    def atMostK(nums, k):
        left = 0
        count = 0
        freq = {}

        for right in range(len(nums)):
            freq[nums[right]] = freq.get(nums[right], 0) + 1

            while len(freq) > k:
                freq[nums[left]] -= 1
                if freq[nums[left]] == 0:
                    del freq[nums[left]]
                left += 1

            count += right - left + 1

        return count

    return atMostK(nums, k) - atMostK(nums, k - 1)
```

### 🔴 LeetCode 424 - 替换后的最长重复字符 ⭐⭐⭐

**难度**：中等 | **类型**：不定长窗口 + 变形

```python
def characterReplacement(s, k):
    left = 0
    max_len = 0
    max_count = 0  # 窗口内出现最多的字符的次数
    freq = {}

    for right in range(len(s)):
        freq[s[right]] = freq.get(s[right], 0) + 1
        max_count = max(max_count, freq[s[right]])

        # 窗口大小 - 最多字符次数 > k，需要收缩
        while right - left + 1 - max_count > k:
            freq[s[left]] -= 1
            left += 1

        max_len = max(max_len, right - left + 1)

    return max_len
```

### 🔴 LeetCode 395 - 至少有 K 个重复字符的最长子串 ⭐⭐⭐⭐

**难度**：中等 | **类型**：分治 + 滑动窗口

```python
def longestSubstring(s, k):
    """分治：以出现次数 < k 的字符为分割点"""

    def helper(s, start, end, k):
        if end - start < k:
            return 0

        freq = {}
        for i in range(start, end):
            freq[s[i]] = freq.get(s[i], 0) + 1

        for mid in range(start, end):
            if freq[s[mid]] < k:
                return max(helper(s, start, mid, k), helper(s, mid + 1, end, k))

        return end - start

    return helper(s, 0, len(s), k)
```

## 代码模板速记

```
定长滑动窗口：
  for right in range(k, n):
      window = window - nums[right-k] + nums[right]

不定长窗口（扩大型）：
  for right in range(n):
      window.add(nums[right])
      while 不满足条件:
          window.remove(nums[left])
          left += 1
      更新结果

不定长窗口（缩小型）：
  for right in range(n):
      window.add(nums[right])
      while 满足条件:
          更新结果
          window.remove(nums[left])
          left += 1

单调队列（窗口最大值）：
  for right in range(n):
      while dq and dq[0] < right - k + 1:
          dq.popleft()
      while dq and nums[dq[-1]] < nums[right]:
          dq.pop()
      dq.append(right)
```
