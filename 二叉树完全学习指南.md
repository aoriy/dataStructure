# 第一部分：二叉树核心概念

## 1.1 二叉树的定义

### 基本定义

**二叉树**是一种每个节点**最多有两个子节点**的树形数据结构。这两个子节点分别称为**左子节点**和**右子节点**。

```
      1           ← 根节点（root）
     / \
    2   3         ← 2 是 1 的左子节点，3 是 1 的右子节点
   / \
  4   5           ← 4 是 2 的左子节点，5 是 2 的右子节点
```

### 核心术语

| 术语 | 定义 | 示例（上图） |
|------|------|-------------|
| **根节点（Root）** | 树的顶端节点 | 1 |
| **叶子节点（Leaf）** | 没有子节点的节点 | 3, 4, 5 |
| **内部节点** | 至少有一个子节点的节点 | 1, 2 |
| **父节点** | 有子节点的节点 | 1 是 2、3 的父节点 |
| **子节点** | 被父节点连接的节点 | 2、3 是 1 的子节点 |
| **兄弟节点** | 同一父节点的节点 | 2 和 3 是兄弟节点 |
| **深度（Depth）** | 从根到该节点的边数 | 节点 4 的深度为 2 |
| **高度（Height）** | 从该节点到最远叶子的边数 | 节点 2 的高度为 1 |
| **层数（Level）** | 深度 + 1 | 节点 4 在第 3 层 |

### 节点定义（Python）

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val      # 节点值
        self.left = left    # 左子节点
        self.right = right  # 右子节点
```

## 1.2 二叉树的分类

### 按形态分类

| 类型 | 定义 | 图例 |
|------|------|------|
| **满二叉树** | 每个节点要么是叶子，要么有两个子节点 | 所有内部节点度数为 2 |
| **完全二叉树** | 除最后一层外，每层节点数达到最大；最后一层节点从左到右排列 | 堆的底层结构 |
| **完美二叉树** | 所有叶子节点在同一层，每个内部节点有两个子节点 | 满二叉树 + 完全二叉树 |
| **平衡二叉树** | 每个节点的左右子树高度差不超过 1 | AVL 树 |
| **二叉搜索树（BST）** | 左子树所有值 < 根 < 右子树所有值 | 用于高效查找 |
| **退化二叉树** | 每个节点最多有一个子节点，退化为链表 | 最坏情况 |

### 图解对比

```
满二叉树：          完全二叉树：        完美二叉树：
      1                 1                 1
     / \               / \               /   \
    2   3             2   3             2     3
   / \               / \               / \   / \
  4   5             4   5             4   5 6   7

平衡二叉树：        二叉搜索树（BST）：   退化二叉树：
      1                 4                 1
     / \               / \                 \
    2   3             2   6                 2
   /                 / \   \                 \
  4                 1   3   7                 3
```

### 特殊二叉树详解

#### 二叉搜索树（BST）

**性质**：对于任意节点，左子树所有值 < 节点值 < 右子树所有值

```
        5
       / \
      3   7
     / \   \
    2   4   8

左子树 [2,3,4] < 5 < 右子树 [7,8] ✅
```

**特点**：
- 中序遍历得到**有序序列**
- 查找、插入、删除平均 O(log n)
- 最坏退化为链表，O(n)

#### 完全二叉树

**性质**：用数组存储时，节点 i 的左子节点在 2i+1，右子节点在 2i+2

```
数组表示：[1, 2, 3, 4, 5, 6, 7]

树形结构：
        1(0)
       /    \
    2(1)    3(2)
   /  \     /
 4(3) 5(4) 6(5)

索引关系：
  节点 1（索引 0）→ 左子 2（索引 2×0+1=1），右子 3（索引 2×0+2=2）
  节点 2（索引 1）→ 左子 4（索引 2×1+1=3），右子 5（索引 2×1+2=4）
```

## 1.3 二叉树的重要性质

### 性质一：节点数与边数的关系

> **n 个节点的二叉树有 n-1 条边**

### 性质二：第 i 层最多有多少节点

> **第 i 层最多有 2^(i-1) 个节点**（i 从 1 开始）

### 性质三：深度为 k 的二叉树最多有多少节点

> **深度为 k 的二叉树最多有 2^k - 1 个节点**

### 性质四：叶子节点数与度为 2 的节点数的关系

> **n₀ = n₂ + 1**（叶子节点数 = 度为 2 的节点数 + 1）

### 性质五：n 个节点的完全二叉树的深度

> **深度 = ⌊log₂n⌋ + 1**

### 性质速查表

| 性质 | 公式 | 说明 |
|------|------|------|
| 边数 | n - 1 | n 为节点数 |
| 第 i 层最大节点数 | 2^(i-1) | i 从 1 开始 |
| 深度 k 的最大节点数 | 2^k - 1 | 完美二叉树 |
| 叶子节点数 | n₂ + 1 | n₂ 为度为 2 的节点数 |
| 完全二叉树深度 | ⌊log₂n⌋ + 1 | 向下取整 |

## 1.4 二叉树的存储结构

### 方式一：链式存储（最常用）

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# 构建二叉树
root = TreeNode(1)
root.left = TreeNode(2)
root.right = TreeNode(3)
root.left.left = TreeNode(4)

```

**优点**：灵活，适合动态操作
**缺点**：需要额外存储指针

### 方式二：数组存储（完全二叉树）

```python
# 完全二叉树用数组存储
tree = [1, 2, 3, 4, 5, 6, 7]

# 索引关系
def left_child(i):
    return 2 * i + 1

def right_child(i):
    return 2 * i + 2

def parent(i):
    return (i - 1) // 2
```

**优点**：节省空间，适合完全二叉树（如堆）
**缺点**：非完全二叉树会浪费空间

# 第二部分：二叉树的遍历

## 2.1 前序遍历

### 定义

**访问顺序**：根节点 → 左子树 → 右子树

```
      1
     / \
    2   3
   / \
  4   5

前序遍历：1 → 2 → 4 → 5 → 3
```

### 递归实现

```python
def preorder(root):
    if not root:
        return
    print(root.val)      # 1. 访问根节点
    preorder(root.left)  # 2. 遍历左子树
    preorder(root.right) # 3. 遍历右子树
```

### 迭代实现（栈）

```python
def preorder_iterative(root):
    if not root:
        return []

    result = []
    stack = [root]

    while stack:
        node = stack.pop()
        result.append(node.val)

        # 注意：先压右子节点，再压左子节点（栈是后进先出）
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)

    return result
```

---

## 2.2 中序遍历

### 定义

**访问顺序**：左子树 → 根节点 → 右子树

```
      1
     / \
    2   3
   / \
  4   5

中序遍历：4 → 2 → 5 → 1 → 3
```

> **重要**：BST 的中序遍历得到**有序序列**

### 递归实现

```python
def inorder(root):
    if not root:
        return
    inorder(root.left)   # 1. 遍历左子树
    print(root.val)      # 2. 访问根节点
    inorder(root.right)  # 3. 遍历右子树
```

### 迭代实现（栈）

```python
def inorder_iterative(root):
    result = []
    stack = []
    current = root

    while current or stack:
        # 一直向左走，直到没有左子节点
        while current:
            stack.append(current)
            current = current.left

        # 弹出栈顶，访问
        current = stack.pop()
        result.append(current.val)

        # 转向右子树
        current = current.right

    return result
```

## 2.3 后序遍历

### 定义

**访问顺序**：左子树 → 右子树 → 根节点

```
      1
     / \
    2   3
   / \
  4   5

后序遍历：4 → 5 → 2 → 3 → 1
```

> **重要**：后序遍历常用于**删除树**、**计算树的高度**、**自底向上处理**

### 递归实现

```python
def postorder(root):
    if not root:
        return
    postorder(root.left)  # 1. 遍历左子树
    postorder(root.right) # 2. 遍历右子树
    print(root.val)       # 3. 访问根节点
```

### 迭代实现（栈 + 反转）

```python
def postorder_iterative(root):
    if not root:
        return []

    result = []
    stack = [root]

    while stack:
        node = stack.pop()
        result.append(node.val)

        # 前序遍历的反向：先压左，再压右
        if node.left:
            stack.append(node.left)
        if node.right:
            stack.append(node.right)

    # 反转结果
    return result[::-1]
```

## 2.4 层序遍历

### 定义

**访问顺序**：从上到下，从左到右，逐层遍历

```
      1
     / \
    2   3
   / \
  4   5

层序遍历：1 → 2 → 3 → 4 → 5
```

### 实现（队列）

```python
from collections import deque

def level_order(root):
    if not root:
        return []

    result = []
    queue = deque([root])

    while queue:
        level_size = len(queue)  # 当前层的节点数
        level = []

        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)

            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)

        result.append(level)  # 按层存储

    return result
```

### 变体：返回一维数组

```python
def level_order_flat(root):
    if not root:
        return []

    result = []
    queue = deque([root])

    while queue:
        node = queue.popleft()
        result.append(node.val)

        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)

    return result
```

## 2.5 遍历方式对比与选择

### 四种遍历对比

| 遍历方式 | 访问顺序 | 数据结构 | 典型应用 |
|---------|---------|---------|---------|
| **前序** | 根→左→右 | 栈 | 复制树、序列化 |
| **中序** | 左→根→右 | 栈 | BST 有序遍历 |
| **后序** | 左→右→根 | 栈 | 删除树、计算高度 |
| **层序** | 逐层 | 队列 | 层级处理、右视图 |

### 如何选择遍历方式？

```
需要按层处理？
├── 是 → 层序遍历
└── 否
    ├── 需要有序序列（BST）？
    │   ├── 是 → 中序遍历
    │   └── 否
    │       ├── 需要先处理根节点？
    │       │   ├── 是 → 前序遍历
    │       │   └── 否 → 后序遍历（自底向上）
```

### 递归 vs 迭代

| 对比维度 | 递归 | 迭代 |
|---------|------|------|
| **代码简洁度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| **空间复杂度** | O(h) 递归栈 | O(h) 显式栈 |
| **栈溢出风险** | 有（深度大时） | 无 |
| **推荐场景** | 一般情况 | 深度大或面试要求 |

# 第三部分：二叉树核心操作与技巧

## 3.1 递归思维：三大遍历框架

### 核心框架

```python
def traverse(root):
    if not root:
        return

    # ===== 前序位置 =====
    # 在进入节点时执行
    # 此时 root 是当前节点，左右子树还未处理

    traverse(root.left)

    # ===== 中序位置 =====
    # 在左子树处理完、右子树未处理时执行
    # 对于 BST，此时 root 是"中间"节点

    traverse(root.right)

    # ===== 后序位置 =====
    # 在离开节点时执行
    # 此时左右子树都已处理完毕
```

### 三种位置的典型应用

| 位置 | 执行时机 | 典型应用 |
|------|---------|---------|
| **前序** | 进入节点时 | 打印、复制、路径记录 |
| **中序** | 左子树处理完后 | BST 操作、表达式求值 |
| **后序** | 离开节点时 | 计算高度、删除、自底向上 DP |

## 3.2 二叉树解题通用模板

### 模板一：求属性（高度、深度、节点数等）

```python
def get_property(root):
    if not root:
        return 0  # 或其他基础值

    left = get_property(root.left)
    right = get_property(root.right)

    # 后序位置：根据左右子树的结果计算当前节点
    return max(left, right) + 1  # 或其他计算
```

### 模板二：判断性质（是否平衡、是否 BST 等）

```python
def is_property(root):
    if not root:
        return True  # 或其他基础情况

    left = is_property(root.left)
    right = is_property(root.right)

    # 判断当前节点是否满足条件
    return left and right and condition(root)
```

### 模板三：路径问题（路径和、路径记录）

```python
def path_problem(root, target, path):
    if not root:
        return

    path.append(root.val)  # 前序：进入节点

    if is_leaf(root) and sum(path) == target:
        result.append(path[:])

    path_problem(root.left, target, path)
    path_problem(root.right, target, path)

    path.pop()  # 后序：离开节点，撤销选择
```

### 模板四：修改树结构（翻转、删除等）

```python
def modify_tree(root):
    if not root:
        return None

    # 先处理子树
    left = modify_tree(root.left)
    right = modify_tree(root.right)

    # 后序位置：修改当前节点
    root.left = right
    root.right = left

    return root
```

## 3.3 核心技巧总结

### 技巧一：自顶向下 vs 自底向上

| 方式 | 特点 | 典型应用 |
|------|------|---------|
| **自顶向下** | 前序位置处理，参数向下传递 | 路径记录、深度计算 |
| **自底向上** | 后序位置处理，结果向上返回 | 高度计算、平衡判断 |

**示例：计算深度**

```python
# 自顶向下（前序）
def max_depth_topdown(root, depth=0):
    if not root:
        return
    max_depth = max(max_depth, depth + 1)
    max_depth_topdown(root.left, depth + 1)
    max_depth_topdown(root.right, depth + 1)

# 自底向上（后序）
def max_depth_bottomup(root):
    if not root:
        return 0
    left = max_depth_bottomup(root.left)
    right = max_depth_bottomup(root.right)
    return max(left, right) + 1
```

### 技巧二：使用全局变量

```python
result = 0  # 全局变量

def dfs(root):
    global result
    if not root:
        return
    result = max(result, some_value)
    dfs(root.left)
    dfs(root.right)
```

### 技巧三：返回多个值

```python
def dfs(root):
    if not root:
        return 0, 0  # 返回多个值

    left_height, left_count = dfs(root.left)
    right_height, right_count = dfs(root.right)

    return max(left_height, right_height) + 1, left_count + right_count + 1
```

### 技巧四：使用辅助函数

```python
def is_balanced(root):
    def height(node):
        if not node:
            return 0
        left = height(node.left)
        right = height(node.right)
        if left == -1 or right == -1 or abs(left - right) > 1:
            return -1  # 不平衡
        return max(left, right) + 1

    return height(root) != -1
```

## 4.1 入门级题目

### 🟢 LeetCode 104 - 二叉树的最大深度 ⭐

**难度**：简单 | **类型**：后序遍历

```python
def maxDepth(root):
    if not root:
        return 0
    return max(maxDepth(root.left), maxDepth(root.right)) + 1
```

### 🟢 LeetCode 111 - 二叉树的最小深度 ⭐

**难度**：简单 | **类型**：后序遍历 + 特殊处理

```python
def minDepth(root):
    if not root:
        return 0

    left = minDepth(root.left)
    right = minDepth(root.right)

    # 如果左子树为空，返回右子树深度 + 1
    if not root.left:
        return right + 1
    # 如果右子树为空，返回左子树深度 + 1
    if not root.right:
        return left + 1

    return min(left, right) + 1
```

### 🟢 LeetCode 100 - 相同的树 ⭐

**难度**：简单 | **类型**：递归比较

```python
def isSameTree(p, q):
    if not p and not q:
        return True
    if not p or not q:
        return False
    return p.val == q.val and isSameTree(p.left, q.left) and isSameTree(p.right, q.right)
```

### 🟢 LeetCode 101 - 对称二叉树 ⭐

**难度**：简单 | **类型**：递归比较

```python
def isSymmetric(root):
    def is_mirror(left, right):
        if not left and not right:
            return True
        if not left or not right:
            return False
        return (left.val == right.val and
                is_mirror(left.left, right.right) and
                is_mirror(left.right, right.left))

    return is_mirror(root.left, root.right) if root else True
```

```python
def lowestCommonAncestor(root, p, q):
    if not root or root == p or root == q:
        return root

    left = lowestCommonAncestor(root.left, p, q)
    right = lowestCommonAncestor(root.right, p, q)

    if left and right:
        return root  # p 和 q 分别在左右子树
    return left or right  # p 和 q 都在一侧
```

```python
def buildTree(preorder, inorder):
    if not preorder or not inorder:
        return None

    root_val = preorder[0]
    root = TreeNode(root_val)

    mid = inorder.index(root_val)

    root.left = buildTree(preorder[1:mid+1], inorder[:mid])
```

```python
def levelOrder(root):
    if not root:
        return []

    result = []
    queue = deque([root])

    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        result.append(level)

    return result
```

### 🟡 LeetCode 98 - 验证二叉搜索树 ⭐⭐⭐

**难度**：中等 | **类型**：中序遍历 / 递归验证

```python
# 方法一：中序遍历（BST 中序是有序的）
def isValidBST(root):
    prev = float('-inf')

    def inorder(node):
        nonlocal prev
        if not node:
            return True
        if not inorder(node.left):
            return False
        if node.val <= prev:
            return False
        prev = node.val
        return inorder(node.right)

    return inorder(root)

# 方法二：递归验证（传递上下界）
def isValidBST(root):
    def validate(node, low, high):
        if not node:
            return True
        if node.val <= low or node.val >= high:
            return False
        return validate(node.left, low, node.val) and validate(node.right, node.val, high)

    return validate(root, float('-inf'), float('inf'))
```

### 🟡 LeetCode 114 - 二叉树展开为链表 ⭐⭐⭐

**难度**：中等 | **类型**：后序遍历 + 修改结构

```python
def flatten(root):
    if not root:
        return

    flatten(root.left)
    flatten(root.right)

    # 后序位置：将左子树接到右边
    temp = root.right
    root.right = root.left
    root.left = None

    # 找到当前链表末尾，接上原来的右子树
    current = root
    while current.right:
        current = current.right
    current.right = temp
```

## 4.3 高级题目

### 🔴 LeetCode 124 - 二叉树中的最大路径和 ⭐⭐⭐⭐

**难度**：困难 | **类型**：后序遍历 + 动态规划

```python
def maxPathSum(root):
    max_sum = float('-inf')

    def dfs(node):
        nonlocal max_sum
        if not node:
            return 0

        left = max(0, dfs(node.left))   # 负数不选
        right = max(0, dfs(node.right))

        # 当前节点作为拐点的路径和
        max_sum = max(max_sum, node.val + left + right)

        # 返回以当前节点为起点的最大路径
        return node.val + max(left, right)

    dfs(root)
    return max_sum
```

```python
class Codec:
    def serialize(self, root):
        def dfs(node):
            if not node:
                return 'None,'
            return str(node.val) + ',' + dfs(node.left) + dfs(node.right)
        return dfs(root)

    def deserialize(self, data):
        def dfs(nodes):
            val = nodes.pop(0)
            if val == 'None':
                return None
            node = TreeNode(int(val))
            node.left = dfs(nodes)
            node.right = dfs(nodes)
            return node

        return dfs(data.split(','))
```

### 🔴 LeetCode 99 - 恢复二叉搜索树 ⭐⭐⭐⭐

```python
def recoverTree(root):
    first = second = prev = None

    def inorder(node):
        nonlocal first, second, prev
        if not node:
            return

        inorder(node.left)

        if prev and prev.val > node.val:
            if not first:
                first = prev
            second = node

        prev = node
        inorder(node.right)

    inorder(root)
    first.val, second.val = second.val, first.val
```

# 第五部分：常见陷阱与对比分析

## 5.1 常见陷阱与易错点

### 陷阱一：混淆深度和高度

| 概念 | 定义 | 计算方式 |
|------|------|---------|
| **深度** | 从根到该节点的边数 | 从上往下累加 |
| **高度** | 从该节点到最远叶子的边数 | 从下往上累加 |

> **根节点的深度为 0，高度为树的高度**

### 陷阱二：最小深度的特殊情况

```python
# ❌ 错误：直接取 min
def minDepth_wrong(root):
    if not root:
        return 0
    return min(minDepth_wrong(root.left), minDepth_wrong(root.right)) + 1

# ✅ 正确：处理单子树的情况
def minDepth_correct(root):
    if not root:
        return 0
    if not root.left:
        return minDepth_correct(root.right) + 1
    if not root.right:
        return minDepth_correct(root.left) + 1
    return min(minDepth_correct(root.left), minDepth_correct(root.right)) + 1
```

### 陷阱三：BST 验证的上下界

```python
# ❌ 错误：只比较父子节点
def isBST_wrong(root):
    if not root:
        return True
    if root.left and root.left.val >= root.val:
        return False
    if root.right and root.right.val <= root.val:
        return False
    return isBST_wrong(root.left) and isBST_wrong(root.right)

# ✅ 正确：传递上下界
def isBST_correct(root, low=float('-inf'), high=float('inf')):
    if not root:
        return True
    if root.val <= low or root.val >= high:
        return False
    return isBST_correct(root.left, low, root.val) and isBST_correct(root.right, root.val, high)
```

### 陷阱四：路径问题的回溯

```python
# ❌ 错误：忘记撤销选择
def pathSum_wrong(root, target, path):
    if not root:
        return
    path.append(root.val)
    if sum(path) == target:
        result.append(path)  # 应该是 path[:]
    pathSum_wrong(root.left, target, path)
    pathSum_wrong(root.right, target, path)
    # 忘记 path.pop()

# ✅ 正确：记录副本 + 撤销选择
def pathSum_correct(root, target, path):
    if not root:
        return
    path.append(root.val)
    if sum(path) == target:
        result.append(path[:])
    pathSum_correct(root.left, target, path)
    pathSum_correct(root.right, target, path)
    path.pop()  # 撤销选择
```

### 陷阱五：递归返回值的使用

```python
# ❌ 错误：忽略返回值
def invertTree_wrong(root):
    if not root:
        return
    invertTree_wrong(root.left)
    invertTree_wrong(root.right)
    root.left, root.right = root.right, root.left

# ✅ 正确：正确使用返回值
def invertTree_correct(root):
    if not root:
        return None
    left = invertTree_correct(root.left)
    right = invertTree_correct(root.right)
    root.left = right
    root.right = left
```

## 二叉树题型识别指南

### 高频信号词

| 信号词 | 可能的解法 |
|--------|-----------|
| "深度"、"高度" | 后序遍历，自底向上 |
| "层"、"层序" | BFS 队列 |
| "路径"、"路径和" | 前序遍历 + 回溯 |
| "公共祖先" | 后序遍历 |
| "BST"、"搜索树" | 中序遍历或递归验证 |
| "构造"、"重建" | 递归分治 |
| "翻转"、"镜像" | 后序遍历修改 |
| "对称"、"相同" | 递归比较 |

### 判断流程

```
题目涉及"遍历"？
├── 是 → 选择合适的遍历方式
└── 否
    ├── 题目涉及"路径"？
    │   ├── 是 → 前序遍历 + 回溯
    │   └── 否
    │       ├── 题目涉及"深度/高度"？
    │       │   ├── 是 → 后序遍历
    │       │   └── 否
    │       │       ├── 题目涉及"BST"？
    │       │       │   ├── 是 → 中序遍历或递归验证
    │       │       │   └── 否 → 考虑其他方法
```
