# 计算机四大件一年系统学习计划

> 适合人群：有 Python 基础的软件测试新手
> 学习周期：12 个月（2025年7月 ~ 2026年6月）
> 终极目标：独立开发完整项目，编写 .exe 程序，具备转岗开发的能力

---

## 总体路线图

```
第1-2月   Python进阶 + 编程基础强化
  ↓       🎯 里程碑：能独立写出 500+ 行的 Python 项目
第3-4月   数据结构与算法
  ↓       🎯 里程碑：能解 150+ 道 LeetCode，掌握常见算法模式
第5-6月   计算机网络 + 数据库
  ↓       🎯 里程碑：能设计数据库表、写 SQL、理解网络协议
第7-8月   操作系统 + Linux
  ↓       🎯 里程碑：能在 Linux 上部署项目，理解 OS 核心概念
第9-10月  计算机组成原理 + GUI/桌面应用开发
  ↓       🎯 里程碑：能开发带界面的桌面应用，打包为 .exe
第11-12月 综合实战项目 + 面试准备
          🎯 里程碑：独立完成 2-3 个完整项目，准备面试
```

## 给你的额外建议

### 1. 发挥你的测试优势

作为软件测试，你有以下天然优势：
- **质量意识**：你会更注重代码质量和边界情况
- **验证思维**：你会自然地写测试用例
- **业务理解**：你更懂用户需求和产品逻辑

### 2. 项目选择建议

每个项目尽量覆盖多个知识点：
- 个人记账本 → GUI + 数据库 + 打包
- API 测试平台 → 网络 + 数据库 + Web + 部署
- 文件处理工具 → GUI + 文件系统 + 算法

### 3. 学习节奏

- **不要贪多**：每个阶段扎实掌握再进入下一阶段
- **不要跳过编程实践**：四大件的理论要通过代码来理解
- **不要孤立学习**：每个知识点都要和实际项目结合

### 4. 面试准备

从第 10 月开始，每周花 1 小时准备面试：
- 整理项目经历（STAR 法则）
- 准备常见面试题回答
- 模拟技术面试
